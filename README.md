- SAR (Sentinel-1) vs optique (Sentinel-2)
- Formats cloud-natifs géospatiaux : COG, Zarr, STAC
- Bases d'orbitographie : LEO/GEO, orbites héliosynchrones
- Cryptographie appliquée : AES, RSA, TLS, PKI
